Dialog

Description:

Dialog panel it's opened using the dialog service. (ex. this.dialogService.open(COMPONENT, customData)

Those are the custom data available for dialog config:
data?: any; | Custom data that you would need to add from outside inside the dialog.
title?: string; | Dialog title
showSaveButton?: boolean; | Option to configure the show/hide of the save button.
showDismissButton?: boolean; | Option to configure the show/hide of the dismiss button.
saveButtonLabel?: string; | Option to configure the save button label.
dismissButtonLabel?: string; | Option to configure the dismiss button label.
width?: string; | Option to configure the width of the dialog.

On each dialog action an event it's emitted: If you click on the X icon, the event emitted it's DialogStatus.CLOSE, if you click on save the event emitted is DialogStatus.SAVE, if you click on dismiss the event it's DialogStatus.DISMISS. By using this action on the actionEvent output event emitter, you can bind your function and listen to events for your custom functionality.

Additionaly, you can change the templates of the header and footer and provide a totally custom implementation as you would need it. Check the "alternative templates" examples
